Operadores Aritméticos:
+ = Adição -> 5 + 2 == 7
- = Subtração -> 5 - 2 == 3
* = Mutiplicação -> 5 * 2 == 10
/ = Divisão -> 5 / 2 == 2.5
** = Potencia -> 5 ** 2 == 25
// = Divisão Inteira -> 5 // 2 == 2
% = Resto da Divisão -> 5 % 2 == 1
